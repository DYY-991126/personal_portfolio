## Objective

Turn aligned context into a complete website generation direction that is ready to guide real implementation.

## Use When

- The user has provided enough context to move from沟通阶段 into generation
- A scenario skill such as maps, redesign, or clone has already collected enough context
- The request is a normal text-only website request

## Design Expectations

- Build a complete page and section plan instead of a vague concept
- Keep the structure demand-driven, minimal, and goal-effective
- Translate the selected style direction into a concrete website system
- Identify what media is needed for the final site, instead of treating visuals as an afterthought
- Produce output that can directly guide subsequent generation or implementation

## Can Invoke

- `image_generation`
- `video_generation`

## Outputs

- Website generation direction
- Page and module system
- Media requirements
- Implementation-ready design brief
