## Skills Overview

- `maps`: Handle Google Maps business sources and prepare them for website generation.
- `redesign`: Handle redesign requests based on an existing website.
- `clone`: Handle close-reference or recreation requests based on an existing website.
- `website_design`: Turn collected context into a complete website generation direction.
- `image_generation`: Generate image-based visual outputs when website design needs them.
- `video_generation`: Generate video-based outputs when website design needs them.
